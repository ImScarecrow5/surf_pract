# Выбор гостей

**ID:** BS-04  
**Приоритет:** Medium  
**Статус:** Готов к дизайну  

---

## Задача дизайнера

> Позволяет добавить гостей (0-2 человека) к бронированию

---

## Визуальная структура

```
┌─────────────────────────────────────┐
│ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ │
├─────────────────────────────────────┤
│         ─────────                   │
│                                     │
│     Добавить гостей                 │
│                                     │
│        [ - ]   1   [ + ]            │  ← Counter
│                                     │
│   Вы можете взять с собой           │
│   до 2 гостей                       │
│                                     │
│   [ Добавить ]                      │
│                                     │
└─────────────────────────────────────┘
```

---

## Элементы

- Заголовок
- Счётчик гостей (0, 1, 2)
- Подсказка
- Кнопка "Добавить"

---

## Взаимодействия (Web Modal)

| Действие | Результат |
|----------|-----------|
| Клик "+" | Увеличить на 1 (макс 2) |
| Клик "-" | Уменьшить на 1 (мин 0) |
| Клик "Добавить" | Сохранить и закрыть модальное окно |
| Клик вне модального окна | Закрыть модальное окно |
| Клавиша Escape | Закрыть модальное окно |
