# Список бронирований

**ID:** SCR-05  
**Приоритет:** High  
**Статус:** Готов к дизайну  

---

## Задача дизайнера

> Краткое описание: пользователь просматривает список своих бронирований ( upcoming и past )

### User Story
> Как **клиент**, я хочу **видеть свои бронирования**, чтобы **не пропустить прогулку**.

### Бизнес-ценность
- Клиент видит все записи в одном месте
- Быстрый доступ к деталям брони

---

## Визуальная структура (WEB)

```
┌─────────────────────────────────────┐
│ Мои бронирования                    │  ← Header
├─────────────────────────────────────┤
│ [Все] [Предстоящие] [Прошедшие]     │  ← Segmented control
├─────────────────────────────────────┤
│                                     │
│  ┌─────────────────────────────────┐│
│  │ 📅 15 июня, 10:00              ││  ← Карточка
│  │ Вдоль Пресненской               ││
│  │ 2 часа · Вы записаны           ││
│  └─────────────────────────────────┘│
│                                     │
│  ┌─────────────────────────────────┐│
│  │ 📅 18 июня, 14:00              ││
│  │ Парк Горького                  ││
│  │ 1.5 часа · Вы записаны         ││
│  └─────────────────────────────────┘│
│                                     │
└─────────────────────────────────────┘
```

## Элементы экрана

### Header

| Элемент | Тип | Описание |
|---------|-----|----------|
| Кнопка «Назад» / Навигация | Кнопка | Возврат на главный экран |
| Заголовок | Text | «Мои бронирования» |

### Основной контент

| Элемент | Компонент | Состояния | Поведение |
|---------|-----------|-----------|-----------|
| Сегменты | Segmented Control | normal / selected | Фильтрация списка |
| Список карточек | List | empty / loaded | Клик → детали |

### Footer

| Элемент | Компонент | Состояния |
|---------|-----------|-----------|
| Нет | — | — |

---

## Состояния экрана

| Состояние | Когда показывается | Визуал |
|-----------|-------------------|--------|
| **Загрузка** | Ожидание данных | Скелетон |
| **Контент** | Данные загружены | Список карточек |
| **Пусто** | Нет бронирований | Иллюстрация + текст |
| **Ошибка** | Ошибка API | Текст + кнопка |

## Взаимодействия (UX)

| Действие | Элемент | Результат |
|----------|---------|-----------|
| Клик | Карточка | Переход к деталям |
| Наведение | Карточка | Подсветка |
