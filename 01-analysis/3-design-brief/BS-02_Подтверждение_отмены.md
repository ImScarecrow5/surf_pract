# Modal подтверждения отмены бронирования

**ID:** BS-04  
**Приоритет:** High  
**Статус:** Черновик  
**Тип элемента:** Web Modal

---

## Задача дизайнера

> Modal с подтверждением отмены бронирования. Пользователь видит предупреждение и условия политики отмены перед подтверждением отмены.

### User Story
> Как **посетитель скалодрома**, я хочу **подтвердить отмену бронирования**, чтобы **отменить запись на занятие с учётом условий политики**.

### Бизнес-ценность
- Информирование пользователя о финансовых последствиях отмены
- Снижение количества импульсивных отмен
- Прозрачность условий политики отмены

---

## Визуальная структура

```
┌─────────────────────────────────────┐
│ ≡                    [×]           │  ← Header (закреплён)
├─────────────────────────────────────┤
│                                     │
│     ⚠️ Вы уверены, что хотите       │  ← Скроллится
│        отменить запись?             │
│                                     │
│  ─────────────────────────────────  │
│                                     │
│     Политика отмены:               │
│     • Бесплатно, если ≥2ч до старта │
│     • Штраф 10%, если <2ч до старта │
│                                     │
│     Сумма к оплате: 500 ₽          │
│                                     │
├─────────────────────────────────────┤
│ [Не отменять]      [Отменить]       │  ← Footer (закреплён)
└─────────────────────────────────────┘
```

## Элементы экрана

### Header

| Элемент | Тип | Описание |
|---------|-----|----------|
| Заголовок | Текст | «Подтверждение отмены» |
| Кнопка закрытия | Иконка [×] | Закрывает Modal |

### Основной контент

| Элемент | Компонент | Состояния | Поведение |
|---------|-----------|-----------|-----------|
| Иконка предупреждения | Иконка | normal | Статическая |
| Текст вопроса | Заголовок | normal | Статический |
| Разделитель | Линия | normal | Визуальный |
| Блок политики отмены | Карточка | normal | Статический |
| Итоговая сумма штрафа | Текст | normal / hidden | Показывается при наличии штрафа |

### Footer

| Элемент | Компонент | Состояния |
|---------|-----------|-----------|
| Кнопка «Не отменять» | Secondary Button | active / disabled |
| Кнопка «Отменить» | Primary Button (красная) | active / disabled / loading |

---

## Состояния экрана

| Состояние | Когда показывается | Визуал |
|-----------|-------------------|--------|
| **Без штрафа** | ≥2 часа до старта | Текст «Бесплатно», кнопка «Отменить» |
| **Со штрафом** | <2 часа до старта | Сумма штрафа, кнопка «Отменить» |
| **Загрузка** | Ожидание ответа API | Спиннер на кнопке «Отменить» |
| **Успех** | Отмена подтверждена | Закрытие Modal, показ снекбара |

## Взаимодействия (UX)

### Навигация

| Откуда | Как открывается |
|--------|-----------------|
| Экран деталей бронирования | По клику на кнопку «Отменить» |
| Deep link | Не поддерживается |

### Действия пользователя

| Действие | Элемент | Результат |
|----------|---------|-----------|
| Клик | Кнопка [×] | Закрытие Modal |
| Клик по оверлею | Область контента | Закрытие Modal |
| Клик | Кнопка «Не отменять» | Закрытие Modal |
| Клик | Кнопка «Отменить» | Отправка запроса на отмену |

### Анимации

| Событие | Анимация | Длительность |
|---------|----------|--------------|
| Появление Modal | Fade in + scale | 200ms |
| Закрытие Modal | Fade out + scale | 150ms |
| Клик кнопки | Scale down + ripple | 150ms |
| Загрузка | Спиннер | Бесконечно |
