# Главный экран

**ID:** SCR-02  
**Приоритет:** High  
**Статус:** Готов к дизайну

---

## Задача дизайнера

> Главный экран приложения с двумя табами: "Расписание" для просмотра доступных слотов и "Мои бронирования" для просмотра бронирований пользователя. Позволяет быстро переключаться между просмотром слотов и управлением бронированиями.

### User Story
> Как **клиент скалодрома**, я хочу иметь доступ к расписанию слотов и моим бронированиям с одного экрана, чтобы удобно планировать посещение и отслеживать свои брони.

### Бизнес-ценность
- Удобный доступ к доступным слотам для бронирования
- Быстрый просмотр и управление своими бронированиями
- Повышение конверсии в бронирования через интуитивный интерфейс

---

## Визуальная структура

```
┌─────────────────────────────────────┐
│ Расписание               [Фильтры]  │  ← Header (закреплён)
├─────────────────────────────────────┤
│                                     │
│        Основной контент            │  ← Скроллится (табы)
│        (список слотов или           │
│         бронирований)               │
│                                     │
├─────────────────────────────────────┤
│ [Расписание]    [Мои бронирования] │  ← Bottom Navigation
└─────────────────────────────────────┘
```

## Элементы экрана

### Header

| Элемент | Тип | Описание |
|---------|-----|----------|
| Заголовок | Текст | Динамический: "Расписание" или "Мои бронирования" |
| Кнопка фильтров | Иконка BS-01 | Открытие модального окна фильтров |

### Bottom Navigation

| Элемент | Тип | Описание |
|---------|-----|----------|
| Таб "Расписание" | Иконка + Текст | Переключение на список слотов |
| Таб "Мои бронирования" | Иконка + Текст | Переключение на список бронирований |
| Индикатор активного таба | Подчёркивание/заливка | Визуальное обозначение активного таба |

### Основной контент — Таб "Расписание"

| Элемент | Компонент | Состояния | Поведение |
|---------|-----------|-----------|-----------|
| Слот | Карточка | normal / disabled (0 мест) | Клик → переход на SCR-03 |
| Дата | Заголовок секции | - | Группировка слотов по датам |
| Индикатор типа | Бейдж | boulder / rope | Для новичков только boulder |
| Индикатор мест | Текст | normal / warning (<3) | Слоты с 0 мест скрыты |
| Пустое состояние | Блок | - | Empty state при отсутствии слотов |
| Скелетон загрузки | Карточки | loading | Показа при загрузке |

### Основной контент — Таб "Мои бронирования"

| Элемент | Компонент | Состояния | Поведение |
|---------|-----------|-----------|-----------|
| Бронирование | Карточка | active / cancelled / cancelled_by_gym | Клик → переход на SCR-06 (SCR-03) |
| Дата и время | Текст | - | Информация о забронированном слоте |
| Статус | Бейдж | active / cancelled / cancelled_by_gym | Визуальное обозначение статуса |
| Пустое состояние | Блок | - | Empty state при отсутствии бронирований |
| Скелетон загрузки | Карточки | loading | Показа при загрузке |

---

## Состояния экрана

| Состояние | Когда показывается | Визуал |
|-----------|-------------------|--------|
| **Загрузка** | Ожидание данных | Скелетон карточек (6-8 шт.) |
| **Контент** | Данные загружены | Список слотов / бронирований |
| **Пусто** | Нет данных | Empty state: иллюстрация + текст |
| **Ошибка** | Ошибка API / сети | Текст ошибки + кнопка повтора |
| **Слоты с 0 мест** | Все слоты заняты | Пустое состояние "Нет свободных слотов" |

## Взаимодействия (UX)

### Навигация

| Откуда | Как открывается |
|--------|-----------------|
| SCR-01 (Онбординг) | Автоматически после successful login |
| Deep link | `climbbook://schedule` или `climbbook://bookings` |

### Действия пользователя

| Действие | Элемент | Результат |
|----------|---------|-----------|
| Клик | Таб "Расписание" | Переключение на список слотов |
| Клик | Таб "Мои бронирования" | Переключение на список бронирований |
| Клик | Кнопка фильтров | Открытие модального окна BS-01 |
| Клик | Слот (таб Расписание) | Переход на SCR-03 |
| Клик | Бронирование | Переход на SCR-06 (SCR-03) |
| Скролл | Контент | Скроллится независимо для каждого таба |

### Анимации

| Событие | Анимация | Длительность |
|---------|----------|--------------|
| Появление экрана | Fade in + slide up | 300ms |
| Переключение табов | Cross-fade контента | 200ms |
| Клик кнопки | Scale down + ripple | 150ms |
| Появление карточек | Staggered fade in | 50ms между карточками |

## Требования к UI

### Сетка и отступы

- Padding экрана: **16px** по бокам
- Между элементами: **8px / 12px / 16px**
- Padding карточек: **12px / 16px**
- Высота Bottom Navigation: **56px** (с безопасной зоной)
- Отступ карточек друг от друга: **12px**

### Типографика

| Элемент | Размер | Вес | Цвет |
|---------|--------|-----|------|
| Заголовок экрана | 20px | SemiBold | #212121 |
| Заголовок секции (дата) | 16px | Medium | #212121 |
| Основной текст | 14px | Regular | #212121 |
| Вторичный текст | 12px | Regular | #757575 |
| Бейдж статуса | 11px | SemiBold | #FFFFFF |
| Кнопка таба | 12px | Medium | #FF6B35 (active) / #757575 (inactive) |

### Цветовая схема

| Назначение | Цвет |
|-----------|------|
| Primary (кнопки, табы) | #FF6B35 |
| Secondary | #757575 |
| Фон | #F5F5F5 |
| Карточки | #FFFFFF |
| Текст основной | #212121 |
| Текст вторичный | #757575 |
| Успех (активное бронирование) | #4CAF50 |
| Ошибка (отменено) | #F44336 |
| Предупреждение (мало мест) | #FF9800 |
| Отменено скалодромом | #9E9E9E |

### Доступность

- Минимальный размер click-зоны: **44×44px**
- Контраст текста: не менее **4.5:1**
- Focus states для клавиатурной навигации
- Поддержка VoiceOver/TalkBack

## Критерии приёмки дизайна

- [ ] Все элементы соответствуют визуальной структуре
- [ ] Все состояния учтены (загрузка, контент, пусто, ошибка)
- [ ] Навигация и переходы описаны
- [ ] Типографика и цвета соответствуют гайдлайнам
- [ ] Адаптивность для экранов 320px - 428px
- [ ] Все интерактивные элементы имеют достаточный размер для клика
- [ ] Слоты с 0 мест скрыты
- [ ] Для новичков отображается только болдеринг
- [ ] Ближайшие бронирования отображаются первыми
