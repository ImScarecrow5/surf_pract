# Детали слота + Бронирование

**ID:** SCR-03  
**Приоритет:** High  
**Статус:** Готов к дизайну

---

## Задача дизайнера

> Объединённый экран для просмотра деталей тренировки и её бронирования. Пользователь видит полную информацию о слоте (дата, время, зона, инструктор, снаряжение, цена, свободные места), выбирает тип снаряжения (своё/напрокат), видит итоговую цену с учётом аренды, имеет 15-минутный таймер для подтверждения записи.

### User Story
> Как **клиент**, я хочу **просмотреть детали тренировки и забронировать её**, чтобы **записаться на нужную тренировку с выбором снаряжения**.

### Бизнес-ценность
- Увеличение конверсии в бронирования за счёт объединения просмотра и записи в один экран
- Прозрачность ценообразования (отдельная строка для аренды снаряжения)
- Снижение количества отказов от бронирования благодаря таймеру

## Элементы экрана

### Header

| Элемент | Тип | Описание |
|---------|-----|----------|
| Кнопка «Назад» | Иконка (стрелка) | Возврат на экран расписания |
| Заголовок | Текст «Детали тренировки» | Статический заголовок |
| Кнопка «Поделиться» | Иконка | Шaring тренировки (опционально) |

### Основной контент

| Элемент | Компонент | Состояния | Поведение |
|---------|-----------|-----------|-----------|
| Дата и время | Блок | normal | Отображается в формате «ДД.ММ, День, ЧЧ:ММ - ЧЧ:ММ» |
| Зона | Блок | normal | Название зала/зоны |
| Инструктор | Карточка | normal / loading | Фото, имя, рейтинг (звёзды + балл) |
| Снаряжение | Список | normal | Перечень доступного оборудования для аренды |
| Свободные места | Блок | normal / warning (<3) | «Мест: X» или «Осталось X мест» |
| Цена тренировки | Блок | normal | Базовая стоимость без аренды |
| Выбор снаряжения | Segmented Control | normal / disabled | Переключатель «Своё» / «Напрокат» |
| Детали аренды | Блок | hidden / visible | Список выбранного снаряжения + цена за единицу |
| Итоговая цена | Блок | normal | «Итого: X₽» с учётом аренды |
| Таймер | Блок | running / warning (<5мин) / expired | Обратный отсчёт 15:00 |

### Footer

| Элемент | Компонент | Состояния |
|---------|-----------|-----------|
| Кнопка «Подтвердить запись» | Primary Button | active / disabled / loading |
| Таймер | Текст + иконка | normal / warning / expired |

## Состояния экрана

| Состояние | Когда показывается | Визуал |
|-----------|-------------------|--------|
| **Загрузка** | Ожидание данных слота | Скелетон карточек (3 шт.) |
| **Контент** | Данные загружены | Полная информация о слоте |
| **Нет мест** | Свободные места = 0 | Блок «Места закончились», кнопкаdisabled |
| **Таймер истёк** | Прошло 15 минут | Кнопка disabled, текст «Время вышло» |
| **Успех** | Бронирование подтверждено | Экран успеха (inline) |
| **Ошибка** | Ошибка API / сети | Текст ошибки + кнопка «Повторить» |

## Взаимодействия (UX)

### Навигация

| Откуда | Как открывается |
|--------|-----------------|
| SCR-02 (Расписание) | По клику на карточку слота |
| Deep link | `fitapp://slot/{slotId}` |

### Действия пользователя

| Действие | Элемент | Результат |
|----------|---------|-----------|
| Клик | Кнопка «Назад» | Возврат на SCR-02 |
| Клик | Segmented Control «Своё» | Скрыть детали аренды, итог = базовая цена |
| Клик | Segmented Control «Напрокат» | Показать детали аренды, итог = базовая + аренда |
| Скролл | Экран | Скролл контента |
| Таймер истекает | 00:00 | Кнопка «Подтвердить запись» становится disabled |
| Клик | Кнопка «Подтвердить запись» | Загрузка → успех (inline) |
| Клик | Кнопка «На главную» (в успехе) | Переход на SCR-01 |

### Анимации

| Событие | Анимация | Длительность |
|---------|----------|--------------|
| Появление экрана | Fade in + slide up | 300ms, ease-out |
| Переключение снаряжения | Fade in/out блока аренды | 200ms, ease-in-out |
| Таймер < 5 минут | Пульсация красным | 1000ms, loop |
| Успешное бронирование | Scale up иконки + fade in текста | 400ms, spring |
| Клик кнопки | Scale 0.95 | 150ms |
| Hover на кнопку | Background color change + slight scale | 200ms |

## Требования к UI

### Сетка и отступы

- Padding экрана: **16px** по бокам
- Между секциями: **24px**
- Между элементами внутри секции: **12px**
- Padding карточек: **16px**
- Border radius карточек: **12px**

### Цветовая схема

| Назначение | Цвет |
|-----------|------|
| Primary (кнопки) | #FF6B35 |

- Минимальный размер click-зоны: **44×44px**

## Критерии приёмки дизайна

- [ ] Все элементы соответствуют визуальной структуре
- [ ] Все состояния учтены (загрузка, контент, нет мест, таймер истёк, успех, ошибка)
- [ ] Навигация и переходы описаны
- [ ] Типографика и цвета соответствуют гайдлайнам
- [ ] Адаптивность для desktop (1200px+) и tablet (768px+)

- [ ] Таймер визуально меняется при < 5 минутах
- [ ] Успешное состояние отображается inline (без перехода на новый экран)
