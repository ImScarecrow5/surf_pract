## Требования к UI

### Сетка и отступы

- Padding экрана: **{16px}** по бокам
- Между элементами: **{8px / 12px / 16px}**
- Padding карточек: **{12px / 16px}**

### Типографика

| Элемент | Размер | Вес | Цвет |
|---------|--------|-----|------|
| Заголовок | {18-20px} | Bold / SemiBold | {Основной} |
| Подзаголовок | {16px} | Medium | {Основной} |
| Основной текст | {14-16px} | Regular | {Основной} |
| Вторичный текст | {12-14px} | Regular | {Вторичный} |
| Кнопка | {14-16px} | SemiBold | {Белый} |

### Цветовая схема

| Назначение | Цвет |
|-----------|------|
| Primary (кнопки) | {Оранжевый #FF6B35} |
| Secondary | {Серый #757575} |
| Фон | {Светло-серый #F5F5F5} |
| Карточки | {Белый #FFFFFF} |
| Текст основной | {Тёмно-серый #212121} |
| Текст вторичный | {Серый #757575} |
| Успех | {Зелёный #4CAF50} |
| Ошибка | {Красный #F44336} |
| Предупреждение | {Оранжевый #FF9800} |

### Доступность

- Минимальный размер тап-зоны: **{44×44px}**
- Контраст текста: не менее **{4.5:1}**
- Focus states для клавиатурной навигации
