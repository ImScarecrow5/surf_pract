# Модальное окно фильтров

**ID:** BS-01  
**Приоритет:** High  
**Статус:** Готов к дизайну

---

## Задача дизайнера

> Создать модальное окно для фильтрации слотов на главном экране веб-приложения скалодрома «Вертикаль». Пользователь открывает окно по клику на кнопку фильтров, выбирает критерии и применяет их для отображения подходящих слотов.

### User Story
> Как **посетитель скалодрома**, я хочу **фильтровать слоты по зоне, дате, наличию мест и инструктору**, чтобы **быстро найти подходящее время и место для тренировки**.

### Бизнес-ценность
- Улучшение пользовательского опыта при поиске слотов
- Увеличение конверсии бронирований за счёт удобного поиска
- Снижение нагрузки на службу поддержки (интуитивный интерфейс)
- Повторные визиты благодаря удобству фильтрации

## Свойства модального окна

| Свойство | Значение |
|----------|----------|
| **Тип элемента** | Модальное окно (центрированное) |
| **Ширина** | 480px (max-width: 100% - 32px) |
| **Высота** | auto, max 80vh с внутренним скроллом |
| **Закрытие по Escape** | Да |
| **Закрытие по клику на backdrop** | Да, клик вне модального окна |
| **Backdrop** | Да, полупрозрачный затемнённый фон (rgba(0,0,0,0.5)) |
| **Блокировка скролла** | Да, скролл контента за модальным окном заблокирован |
| **Corner radius** | 16px |
| **Animation in** | Fade in + scale (0.95 → 1), 200ms, ease-out |
| **Animation out** | Fade out + scale, 150ms, ease-in |

## Элементы экрана

### Header

| Элемент | Тип | Описание |
|---------|-----|----------|
| Заголовок | Текст «Фильтры» | 18px, SemiBold, цвет #212121 |
| Счётчик активных | Бейдж | Количество активных фильтров, оранжевый |
| Кнопка закрытия | Иконка (✕) | Закрытие модального окна, 44×44px |

### Основной контент

| Элемент | Компонент | Состояния | Поведение |
|---------|-----------|-----------|-----------|
| Фильтр «Зона» | Segmented Control | normal / selected | Переключение: Все / Болдеринг / Трассы с верёвкой |
| Фильтр «Дата» | Segmented Control + Date Picker | normal / selected / expanded | Выбор: Сегодня / Завтра / Эта неделя / Выбрать дату |
| Фильтр «Места» | Checkbox | unchecked / checked | Переключение: только со свободными местами |
| Фильтр «Инструктор» | Radio List | unchecked / checked / loading | Выбор одного инструктора, отображение рейтинга |
| Список инструкторов | List Item | normal / selected / disabled | Клик — выбор инструктора |

### Footer

| Элемент | Компонент | Состояния |
|---------|-----------|-----------|
| Кнопка «Сбросить» | Text Button | active / disabled (если нет активных фильтров) |
| Кнопка «Применить» | Primary Button | active / disabled (если нет слотов по фильтрам) |

## Состояния экрана

| Состояние | Когда показывается | Визуал |
|-----------|-------------------|--------|
| **Загрузка** | Ожидание списка инструкторов | Скелетон списка (3 строки) |
| **Контент** | Данные загружены | Полный интерфейс фильтров |
| **Пусто (инструкторы)** | Нет доступных инструкторов | Текст «Нет доступных инструкторов» |
| **Нет результатов** | Нет слотов по выбранным фильтрам | Показывается после применения: «Нет доступных слотов» |

| Клик | Кнопка «Сбросить» | Сброс всех фильтров в начальное состояние |
| Клик | Кнопка «Применить» | Применение фильтров, закрытие модального окна, обновление списка слотов |

### Анимации

| Событие | Анимация | Длительность |

| Закрытие модального окна | Fade out + scale | 150ms, ease-in |
| Переключение сегментов | Background slide | 200ms |

| Изменение чекбокса | Check mark animation | 200ms |

## Требования к UI

### Сетка и отступы

- Padding экрана: **16px** по бокам
- Padding секций: **16px** сверху и снизу
- Между элементами: **12px**
- Padding карточек: **16px**
- Отступ заголовка секции: **8px** сверху

- Минимальный размер клик-зоны: **44×44px**
