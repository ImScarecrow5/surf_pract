# Профиль + История + Настройки

**ID:** SCR-04  
**Приоритет:** High  
**Статус:** Готов к дизайну

---

## Задача дизайнера

> Экран профиля клиента с тремя секциями (табами): Профиль, История посещений, Настройки уведомлений. Пользователь управляет своими данными, просматривает историю тренировок и настраивает уведомления.

### User Story
> Как **клиент**, я хочу **просматривать свой профиль, историю посещений и настраивать уведомления**, чтобы **контролировать свои данные и не пропускать тренировки**.

### Бизнес-ценность
- Повышение вовлечённости клиентов через прозрачность истории посещений
- Снижение пропусков тренировок через напоминания
- Формирование доверия через отображение статистики посещений

---

## Визуальная структура

```
┌─────────────────────────────────────┐
│ ← Профиль                  [Иконка] │  ← Header (закреплён)
├─────────────────────────────────────┤
│                                     │
│  [Профиль] [История] [Настройки]   │  ← Табы (скролл или фикс)

│        Основной контент            │  ← Скроллится
│                                     │
│                                     │
├─────────────────────────────────────┤
│                                     │  ← Footer (опционально)
└─────────────────────────────────────┘
```

## Элементы экрана

### Header

| Элемент | Тип | Описание |
|---------|-----|----------|
| Заголовок | Текст | «Профиль» |
| Меню / Настройки | Иконка | Переход к дополнительным настройкам (опционально) |
| Ссылка «На главную» | Ссылка/Иконка | Возврат на главную страницу (опционально) |

### Табы

| Элемент | Тип | Описание |
|---------|-----|----------|
| Таб «Профиль» | Таб | Секция с данными пользователя |
| Таб «История» | Клик | Секция с историей посещений |
| Таб «Настройки» | Клик | Секция с переключателями уведомлений |

### Секция «Профиль»

| Элемент | Компонент | Состояния | Поведение |
|---------|-----------|-----------|-----------|
| Аватар | Circle Image | normal / loading | Опционально тап для смены |
| Имя клиента | Text | normal | Отображается из профиля |
| Номер телефона | Text | normal | Отображается маской +7 (XXX) XXX-XX-XX |
| Кнопка «Выйти» | Secondary Button | normal / pressed / loading | Выход из аккаунта → на экран входа |

### Секция «История»

| Элемент | Компонент | Состояния | Поведение |
|---------|-----------|-----------|-----------|
| Статистика | Блок (карточка) | normal | Всего / Посещено / Отменено |
| Список тренировок | List | normal / loading / empty | Скроллится |
| Карточка тренировки | Card | normal | Отображает дату, время, статус |
| Статус «Посещено» | Badge | success | Зелёный индикатор |
| Статус «Отменено» | Badge | error | Красный индикатор |
| Статус «Не был» | Badge | warning | Оранжевый индикатор |

### Секция «Настройки»

| Элемент | Компонент | Состояния | Поведение |
|---------|-----------|-----------|-----------|
| Переключатель «Напоминание за 1 час» | Switch | on / off | Включение/выключение |
| Переключатель «Уведомление об отмене» | Switch | on / off | Включение/выключение |

### Статистика (История)

| Элемент | Компонент | Состояния |
|---------|-----------|-----------|
| Всего тренировок | Number | normal |
| Посещено | Number + Badge | normal |
| Отменено | Number + Badge | normal |

## Состояния экрана

| Состояние | Когда показывается | Визуал |
|-----------|-------------------|--------|
| **Загрузка** | Ожидание данных профиля | Скелетон на всех элементах |
| **Контент** | Данные загружены | Полное отображение |
| **Пусто (История)** | Нет посещённых тренировок | Иллюстрация + «У вас пока нет посещений» |
| **Ошибка** | Ошибка API / сети | Текст ошибки + кнопка «Повторить» |

## Требования к UI

### Сетка и отступы

- Padding экрана: **16px** по бокам
- Между элементами: **8px / 12px / 16px**
- Padding карточек: **16px**
- Padding секций: **16px**

### Типографика

| Элемент | Размер | Вес | Цвет |
|---------|--------|-----|------|
| Заголовок экрана | 20px | SemiBold | #212121 |
| Заголовок секции | 18px | SemiBold | #212121 |
| Имя клиента | 18px | SemiBold | #212121 |
| Телефон | 16px | Regular | #757575 |
| Текст карточки | 14px | Regular | #212121 |
| Вторичный текст | 12px | Regular | #757575 |
| Кнопка | 14px | SemiBold | #FFFFFF / #FF6B35 |

| Назначение | Цвет |
|-----------|------|
| Primary (кнопки) | #FF6B35 |

| Фон | #F5F5F5 |

- Минимальный размер зоны клика: **44×44px**
