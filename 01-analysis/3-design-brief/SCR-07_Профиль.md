# Профиль пользователя

**ID:** SCR-07  
**Приоритет:** Medium  
**Статус:** Готов к дизайну  

---

## Задача дизайнера

> Пользователь просматривает и редактирует свой профиль, настройки уведомлений, историю

### User Story
> Как **клиент**, я хочу **управлять профилем и настройками**, чтобы **контролировать свои данные**.

---

## Визуальная структура

```
┌─────────────────────────────────────┐
│           Профиль                   │  ← Header
├─────────────────────────────────────┤
│                                     │
│         [АВАТАР]                    │
│         Иван Иванов                 │
│         +7 (999) 000-00-00         │
│                                     │
│   [ Редактировать профиль ]         │
│                                     │
├─────────────────────────────────────┤
│   Мои бронирования            >    │
│   Избранные маршруты          >    │
│   История бронирований        >    │
├─────────────────────────────────────┤
│   Настройки                      >  │
│   Уведомления                    >  │
│   Помощь                        >   │
├─────────────────────────────────────┤
│   О сайте                         >   │
│                                     │
│   [ Выйти ]                          │
└─────────────────────────────────────┘
```

---

## Элементы

- Аватар и имя
- Кнопка редактирования
- Секция «Мои бронирования»
- Секция «Настройки»
- Секция «О сайте»
- Кнопка «Выйти»

---

## Состояния

- Контент
- Загрузка (при выходе)

---

## Требования

- Взаимодействие: Клик (не Тап)
- Padding: 16px
- Primary: #FF6B35
- Разделители между секциями
