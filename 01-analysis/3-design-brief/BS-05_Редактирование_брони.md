# Редактирование брони

**ID:** BS-05  
**Приоритет:** Medium  
**Статус:** Готов к дизайну  

---

## Задача дизайнера

> Позволяет изменить количество гостей или добавить комментарий к брони

---

## Визуальная структура

```
┌─────────────────────────────────────┐
│ ░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░ │
├─────────────────────────────────────┤
│         ─────────                   │
│                                     │
│     Изменить бронь                  │
│                                     │
│   Количество гостей                 │
│        [ - ]   2   [ + ]            │
│                                     │
│   Комментарий                       │
│   ┌─────────────────────────────┐  │
│   │ (опционально)               │  │
│   └─────────────────────────────┘  │
│                                     │
│   [ Сохранить ]                     │
│   [ Отменить бронь ]                │  ← Destructive
│                                     │
└─────────────────────────────────────┘
```

---

## Элементы

- Заголовок
- Счётчик гостей
- Поле комментария
- Кнопка "Сохранить"
- Кнопка "Отменить бронь" (красная)

---

## Взаимодействия

| Действие | Результат |
|----------|-----------|
| Клик "+/-" | Изменение количества |
| Клик "Сохранить" | Сохранение + закрытие модального окна |
| Клик "Отменить бронь" | Переход к BS-02 |
