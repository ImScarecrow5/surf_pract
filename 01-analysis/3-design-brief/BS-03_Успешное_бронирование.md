## Задача дизайнера

> Показывается после успешного бронирования слота. Подтверждает запись и даёт следующие шаги.

## Визуальная структура

```
┌─────────────────────────────────────┐
│                               [X]   │  ← Close button
├─────────────────────────────────────┤
│                                     │
│         ✅                           │  ← Icon
│                                     │
│     Вы записаны!                    │  ← Title
│                                     │
│   15 июня, 10:00                   │
│   Вдоль Пресненской                 │
│   Место сбора: Причал №3           │
│                                     │
│   [ К деталям ]                     │  ← Primary Button
│   [ На главную ]                    │  ← Text Button
│                                     │
└─────────────────────────────────────┘
```

- Заголовок "Вы записаны!"
