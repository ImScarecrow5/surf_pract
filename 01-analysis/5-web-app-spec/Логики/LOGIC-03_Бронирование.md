## Обзор

Процесс создания брони включает выбор слота, выбор типа снаряжения (своё или аренда), при необходимости выбор конкретного снаряжения, подтверждение брони и получение подтверждения с временем на оплату.

### User Story

> Как клиент скалодрома, я хочу забронировать тренировку в несколько шагов (выбор слота → снаряжение → подтверждение), чтобы быстро и удобно оформить бронирование.

### Бизнес-ценность

- Понятный процесс бронирования в несколько шагов
- Выбор аренды снаряжения
- Прозрачное отображение итоговой цены
- Подтверждение брони с таймером

## Описание логики

### Шаг 1: Выбор слота и переход к бронированию

Пользователь выбирает слот из списка и переходит к деталям. Если есть свободные места — активна кнопка "Забронировать".

### Шаг 2: Выбор типа снаряжения

На экране выбора снаряжения пользователь выбирает:
- "Своё снаряжение" — без дополнительной оплаты
- "Аренда" — переход к списку снаряжения

### Шаг 3: Выбор снаряжения (при аренде)

При выборе аренды показывается список доступного снаряжения с ценами и количеством. Пользователь выбирает нужное.

### Шаг 4: Подтверждение брони

Экран подтверждения показывает:
- Дату и время тренировки
- Зону
- Инструктора
- Выбранное снаряжение
- Итоговую цену

### Шаг 5: Создание брони

Отправляется POST /bookings с:
- slotId: UUID слота
- equipmentType: "own" или "rental"
- equipmentId: UUID снаряжения (если rental)

### Шаг 6: Подтверждение

При успехе показывается экран успеха с:
- Номером бронирования
- Временем на подтверждение (15 минут)
- Деталями брони

## API запросы

### GET /equipment

**Триггер:** Переход к выбору аренды снаряжения

**Headers:**

| Поле | Описание |
|------|----------|
| authorization | Bearer токен (опционально) |
| deviceuuid | ID устройства |

**Параметры:** Нет

**Обработка ответа:**

| Результат | Действие |
|-----------|----------|
| Успех (200) | Отобразить список снаряжения с ценами |
| Ошибка | Показать пустой список аренды |

---

### POST /bookings

**Триггер:** Тап "Подтвердить"

**Headers:**

| Поле | Описание |
|------|----------|
| authorization | Bearer токен |
| deviceuuid | ID устройства |
| Content-Type | application/json |

**Параметры/Body:**

| Параметр | Тип | Описание | Значение/Источник |
|----------|-----|----------|-------------------|
| slotId | string | UUID слота | Из выбранного слота |
| equipmentType | string | own/rental | Из экрана выбора |
| equipmentId | string | UUID снаряжения | Из списка (если rental) |

**Обработка ответа:**

| Результат | Действие |
|-----------|----------|
| Загрузка | Блокировка кнопки, лоадер |
| Успех (201) | Переход на экран подтверждения |
| Ошибка 400 | Снек с текстом валидации |
| Ошибка 401 | Переход на авторизацию |
| Ошибка 409 | Снек "Нет свободных мест", возврат к списку |
| Ошибка 410 | Снек "Тренировка отменена", возврат к списку |
| Ошибка 5xx | Снек "Ошибка. Попробуйте позже" |

## Критерии приёмки

| ID | Критерий |
|----|----------|
| AC-001 | **Дано** Пользователь выбрал слот, **Когда** Он нажимает "Забронировать", **Тогда** Открывается экран выбора снаряжения |
| AC-002 | **Дано** Пользователь выбрал "Своё снаряжение", **Когда** Он нажимает "Продолжить", **Тогда** Показывается экран подтверждения с ценой без аренды |
| AC-003 | **Дано** Пользователь выбрал "Аренда", **Когда** Он выбирает снаряжение, **Тогда** Цена на экране подтверждения включает стоимость аренды |
| AC-004 | **Дано** Пользователь подтверждает бронирование, **Когда** Нажимает "Подтвердить", **Тогда** Бронирование создаётся и показывается экран успеха |
| AC-005 | **Дано** Бронирование создано, **Когда** Прошло 15 минут без оплаты, **Тогда** Бронирование автоматически отменяется |
| AC-006 | **Дано** Нет свободных мест, **Когда** Пользователь пытается забронировать, **Тогда** Показывается ошибка и кнопка неактивна |

## Обработка ошибок

| Тип ошибки | Контекст | Действие |
|------------|----------|----------|
| Нет мест | Создание брони | Снек, вернуть к списку слотов |
| Слот отменён | Создание брони | Снек, вернуть к списку |
| Валидация | Создание брони | Снек с текстом ошибки |
| Таймаут 15 мин | Подтверждение | Автоотмена брони |
| Нет сети | Любой запрос | Снек "Нет соединения" |
