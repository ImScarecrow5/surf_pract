## Обзор

Логика работы системы фильтров на главном экране приложения. Фильтры позволяют пользователю находить нужные тренировки по зоне, дате и инструктору. Фильтры учитывают уровень пользователя (для новичков — только болдеринг).

### User Story

> Как клиент скалодрома, я хочу фильтровать тренировки по зоне, дате и инструктору на главном экране, чтобы быстро находить подходящие мне занятия.

### Бизнес-ценность

- Интуитивная фильтрация тренировок
- Персонализация под уровень пользователя
- Сохранение фильтров между сессиями
- Мгновенное обновление результатов

## Описание логики

### Шаг 1: Инициализация фильтров

При загрузке главного экрана:
- Если пользователь новичок (beginner) — по умолчанию выбрана зона "Болдеринг"
- Для остальных — все зоны
- Дата: по умолчанию ближайшие 7 дней

### Шаг 2: Загрузка справочных данных

Параллельно загружаются:
- Список зон (/zones)
- Список инструкторов (/instructors)

### Шаг 3: Применение фильтров

При изменении любого фильтра:
- Валидация (новичок не может выбрать трассы)
- Сохранение состояния
- Автоматическая перезагрузка слотов

### Шаг 4: Сохранение состояния

Текущие фильтры сохраняются в локальный кэш для восстановления при возврате на экран.

### Шаг 5: Сброс фильтров

Кнопка "Сбросить" очищает все фильтры и возвращает к состоянию по умолчанию.

## API запросы

### GET /zones

**Триггер:** Загрузка главного экрана (один раз)

**Headers:**

| Поле | Описание |
|------|----------|
| authorization | Bearer токен (опционально) |
| deviceuuid | ID устройства |

**Параметры:** Нет

**Обработка ответа:**

| Результат | Действие |
|-----------|----------|
| Успех (200) | Заполнить выпадающий список зон |
| Ошибка | Использовать дефолтные: Boulder, Трассы |

---

### GET /instructors

**Триггер:** Загрузка главного экрана (один раз)

**Headers:**

| Поле | Описание |
|------|----------|
| authorization | Bearer токен (опционально) |
| deviceuuid | ID устройства |

**Параметры:** Нет

**Обработка ответа:**

| Результат | Действие |
|-----------|----------|
| Успех (200) | Заполнить список инструкторов с рейтингами |
| Ошибка | Показать пустой список |

---

### GET /slots (с фильтрами)

**Триггер:** Любое изменение фильтра

**Headers:**

| Поле | Описание |
|------|----------|
| authorization | Bearer токен |
| deviceuuid | ID устройства |

**Параметры:**

| Параметр | Тип | Описание | Значение/Источник |
|----------|-----|----------|-------------------|
| zone | string | Зона | Из фильтра (boulder/rope/all) |
| date | string | Конкретная дата | Из фильтра даты |
| dateFrom | string | Начало диапазона | При выборе диапазона |
| dateTo | string | Конец диапазона | При выборе диапазона |
| instructorId | string | Инструктор | Из фильтра |
| onlyAvailable | boolean | Только свободные | Всегда true |

**Обработка ответа:**

| Результат | Действие |
|-----------|----------|
| Загрузка | Показать индикатор обновления |
| Успех (200) | Обновить список слотов |
| Ошибка | Снек, показать предыдущие данные |

## Критерии приёмки

| ID | Критерий |
|----|----------|
| AC-001 | **Дано** Пользователь-новичок открывает главный экран, **Когда** Загружаются слоты, **Т_then** По умолчанию выбрана зона "Болдеринг" |
| AC-002 | **Дано** Пользователь-новичок выбирает зону "Трассы", **Когда** Он пытается применить, **Т_then** Показывается подсказка "Для новичков только болдеринг" |
| AC-003 | **Дано** Пользователь выбирает дату, **Когда** Он применяет фильтр, **Т_then** Показываются слоты только на эту дату |
| AC-004 | **Дано** Пользователь выбирает инструктора, **Когда** Он применяет фильтр, **Т_then** Показываются слоты только этого инструктора |
| AC-005 | **Дано** Несколько фильтров активны, **Когда** Пользователь нажимает "Сбросить", **Т_then** Все фильтры очищаются, показываются все слоты |
| AC-006 | **Дато** Фильтры применены, **Когда** Пользователь уходит с экрана и возвращается, **Т_then** Фильтры восстанавливаются |
| AC-007 | **Дано** Нет слотов по выбранным фильтрам, **Когда** Применяются фильтры, **Т_then** Показывается состояние "Нет доступных тренировок" |

## Обработка ошибок

| Тип ошибки | Контекст | Действие |
|------------|----------|----------|
| Новичок выбрал трассы | Фильтр зоны | Показать подсказку, вернуть предыдущее значение |
| Нет результатов | Применение фильтров | Показать состояние пустоты |
| Ошибка загрузки | Загрузка зон/инструкторов | Использовать кэш или дефолтные значения |
| Нет сети | Применение фильтров | Снек "Нет соединения", показать кэш |
