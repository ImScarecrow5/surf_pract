## Обзор

Подробный экран для просмотра информации о слоте тренировки и его бронирования. Клиент может изучить данные о слоте (дата, время, зона, инструктор, оборудование, цена), выбрать тип снаряжения (своё/аренда), посмотреть доступное оборудование и оформить бронирование с таймером 15-минутного подтверждения.

### User Story

> Как **клиент**, я хочу **посмотреть детали слота тренировки и оформить бронирование с выбором снаряжения**, чтобы **забронировать тренировку с нужным снаряжением**.

### Бизнес-ценность

- Увеличение конверсии в бронирования
- Продажа аренды снаряжения
- Прозрачное отображение информации о тренировке

## Используемые запросы

> Все API-запросы экрана с полным описанием параметров и обработки ответов.

### /slots/{slotId}

**Тип:** REST  
**Метод:** GET  
**Спецификация:** [openapi.yaml](../api/openapi.yaml) → `getSlotById`

**Триггер:** Инициализация / Pull-to-refresh

**Параметры:**

| Параметр | Тип | Обязательность | Источник | Описание |
|----------|-----|----------------|----------|----------|
| `slotId` | path (string) | Да | URL | ID слота |

**Обработка ответа:**

| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Загрузка | — | Скелетон карточки (3 сек.) |
| Успех | 200 OK + data | Отобразить данные слота |
| HTTP 404 | Слот не найден | Error state "Слот не найден" |
| HTTP 410 | Слот отменён | Error state "Слот отменён" |
| Сеть | Нет соединения | Error state с кнопкой "Обновить" |

### POST /bookings (Создание бронирования)

**Тип:** REST  
**Метод:** POST  
**Спецификация:** [openapi.yaml](../api/openapi.yaml) → `createBooking`

**Триггер:** Тап на кнопку "Забронировать"

**Параметры:**

| Параметр | Тип | Обязательность | Источник | Описание |
|----------|-----|----------------|----------|----------|
| `slotId` | body (string) | Да | URL / Состояние | ID слота |
| `equipmentType` | body (string) | Да | Состояние | own или rental |
| `equipmentId` | body (string) | Нет | Состояние | ID оборудования (если rental) |

**Обработка ответа:**

| Результат | Условие | UI-реакция |
|-----------|---------|------------|
| Загрузка | — | Лоадер на кнопке, кнопка заблокирована |
| Успех | 201 Created | Показать экран успеха |
| HTTP 400 | Ошибка валидации | Снек с текстом из `message` |
| HTTP 409 | Нет свободных мест | Снек "Нет свободных мест" |
| HTTP 410 | Слот отменён | Error state "Слот отменён" |
| Сеть | Нет соединения | Снек "Нет соединения" |

## Элементы экрана

> **Примечания:**
> - **Колонка "Валидация":** Для полей ввода указать правило и текст ошибки. Для остальных элементов — "—".
> - **Логика:** Описывается после таблицы каждого блока в виде текстового блока "**Логика:**". Если элемент использует переиспользуемую логику из раздела [Логики](Логики/_INDEX.md), укажите ссылку на неё.
> - **Условия доступности:** Для кнопок и интерактивных элементов указать условия активности/видимости после таблицы.

### 1. Header

| Элемент | Описание | Источник данных | Валидация | Действие |
|---------|----------|-----------------|-----------|----------|
| Кнопка "Назад" | Стрелка влево | — | — | Возврат на SCR-02 |
| Заголовок | "Детали тренировки" | — | — | — |
| Кнопка "Избранное" | Иконка сердца | — | — | Добавить в избранное (опционально) |

**Условия доступности:**
- Кнопка "Избранное" опциональна

### 2. Информация о слоте

| Элемент | Описание | Источник данных | Валидация | Действие |
|---------|----------|-----------------|-----------|----------|
| Дата и время | Заголовок | `startTime`, `endTime` из [/slots/{slotId}](#slotsslotid) | — | — |
| Зона | Текст | `zone.name` | — | — |
| Инструктор | Карточка | `instructor` | — | — |
| Доступные места | Бейдж | `freePlaces` | — | Цвет: normal/warning (<3) |
| Цена | Заголовок | `price` | — | — |

**Логика:**
- Инструктор: Если есть, показать имя и фото
- Доступные места: При `freePlaces < 3` показать предупреждение оранжевым

### 3. Выбор снаряжения

| Элемент | Описание | Источник данных | Валидация | Действие |
|---------|----------|-----------------|-----------|----------|
| Segmented Control | "Своё" / "Аренда" | `equipmentType` | — | Переключение |
| Список оборудования | Карточки | `equipment` из [/equipment](#equipment) | — | Выбор конкретного |
| Доплата за аренду | Текст | `pricePerSlot` | — | — |

**Логика:**
- Segmented Control: При выборе "Аренда" показать список оборудования
- Цена = базовая цена + (цена аренды при выборе rental)

**Условия доступности:**
- Segmented Control "Аренда" скрыт, если нет доступного оборудования

### 4. Футер с таймером

| Элемент | Описание | Источник данных | Валидация | Действие |
|---------|----------|-----------------|-----------|----------|
| Таймер | 15:00 обратный отсчёт | Состояние таймера | — | — |
| Итого | Итоговая цена | `price` + `equipment.pricePerSlot` | — | — |
| Кнопка "Забронировать" | Primary Button | — | — | Создать бронирование |

**Логика:**
- Таймер: Запускается при открытии экрана, 15 минут
- При таймер < 5 минут: мигание красным цветом
- При таймер = 0: кнопка "Забронировать" заблокирована

**Условия доступности:**
- Кнопка "Забронировать" активна, если: таймер не истёк И свободные места > 0
- Кнопка "Забронировать" заблокирована при `freePlaces = 0`

## Критерии приёмки

### Позитивные сценарии

| ID | Критерий | Приоритет |
|----|----------|-----------|
| AC-001 | **Дано** авторизованный пользователь, **Когда** открытие экрана, **Тогда** загружаются детали слота | P0 |
| AC-002 | **Дано** слот загружен, **Тогда** отображаются дата, время, зона, инструктор, цена | P0 |
| AC-003 | **Дано** выбор "Аренда", **Тогда** показывается список оборудования | P0 |
| AC-004 | **Дано** корректная цена, **Когда** выбор снаряжения, **Тогда** итоговая цена обновляется | P0 |
| AC-005 | **Дано** тап "Забронировать", **Тогда** создаётся бронирование | P0 |
| AC-006 | **Дано** успешное бронирование, **Тогда** показывается экран успеха | P0 |

### Негативные сценарии

| ID | Критерий | Приоритет |
|----|----------|-----------|
| AC-N01 | **Дано** ошибка сети, **Когда** открытие экрана, **Тогда** отображается error state | P0 |
| AC-N02 | **Дано** свободных мест = 0, **Тогда** кнопка "Забронировать" заблокирована | P0 |
| AC-N03 | **Дано** таймер истёк, **Тогда** кнопка "Забронировать" заблокирована | P0 |
| AC-N04 | **Дано** слот отменён (410), **Тогда** показывается сообщение "Слот отменён" | P0 |

### Граничные условия (Edge Cases)

| ID | Критерий | Приоритет |
|----|----------|-----------|
| AC-E01 | **Дано** таймер < 5 минут, **Тогда** таймер мигает красным | P1 |
| AC-E02 | **Дано** смена типа снаряжения, **Тогда** пересчёт цены без запроса | P1 |
| AC-E03 | **Дано** pull-to-resresh, **Тогда** таймер сбрасывается | P2 |
