# Крутые фичи приложения «Вертикаль»

## Фича 1: Умная система слотов 🔥

**Описание:**  
Автоматическая система бронирования тренировок с умной фильтрацией.

**Что умеет:**
- Показывает только доступные слоты (скрывает полные)
- Для новичков автоматически показывает только болдеринг
- Фильтрация по зоне, дате, инструктору
- Группировка слотов по дате

**Почему круто:**
- Не нужно листать занятые тренировки
- Новичок сразу видит только подходящие занятия
- Один клик до брони

---

## Фича 2: Умная отмена со штрафами ⚖️

**Описание:**  
Автоматический расчёт штрафа при отмене брони.

**Логика:**
- Отмена за 2+ часа до тренировки = **бесплатно**
- Отмена менее чем за 2 часа = **штраф 10%**
- Автоматический расчёт суммы возврата

**Почему круто:**
- Прозрачная система — клиент видит сумму до подтверждения
- Справедливая политика: никто не теряет деньги зря
- Автоматизация исключает ручные ошибки

---

## Фича 3: Визуальные бейджи статусов 🎨

**Описание:**  
Понятная визуальная индикация статуса бронирования с цветовыми бейджами.

**Статусы:**
- `confirmed` — зелёный, тренировка подтверждена
- `cancelled_by_client_early` — красный (бесплатная отмена)
- `cancelled_by_client_late` — красный (штраф 10%)
- `cancelled_by_gym` — серый (отменено скалодромом)

**Почему круто:**
- Мгновенно виден статус бронирования
- Причина отмены понятна сразу
- Сумма штрафа показана перед подтверждением отмены

---

*Все фичи работают через существующую инфраструктуру API — приложение лишь отображает данные и передаёт действия пользователя.*
